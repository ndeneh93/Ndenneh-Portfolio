## About

This portfolio is designed to present a professional online presence for Ndenneh Chubu. It showcases skills, experience, projects, and achievements in an accessible and visually appealing way.

## Features

* **Dark/Light Mode** toggle for better accessibility
* Fully **responsive design** for desktop, tablet, and mobile devices
* Smooth **scroll animations** using AOS (Animate On Scroll)
* **Projects section** with GitHub links and live demos
* **Timeline design** for education, internships, and awards
* Contact form integrated with **Formspree** for messaging

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* [Bootstrap 5](https://getbootstrap.com/)
* [Font Awesome](https://fontawesome.com/)
* [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)

## Installation

To run this portfolio locally:

1. Clone the repository:

   ```bash
   git clone https://github.com/ndeneh93/portfolio.git
   ```
2. Navigate to the project directory:

   ```bash
   cd portfolio
   ```
3. Open `index.html` in your preferred browser.

No additional dependencies are required.

## Usage

* Navigate through sections using the **navbar**
* Toggle between **light and dark mode** using the button on the top-right
* Explore **projects** and click GitHub or Live Demo links to view the work
* Use the **contact form** to send messages directly via Formspree

## Sections
